Welcome to Liver Disease Prediction System

"An AI-powered solution for accurate and early liver disease detection."

Overview

Our Liver Disease Prediction Model is an advanced AI-powered system designed to analyze patient data and predict liver disease with high accuracy. By leveraging multiple machine learning algorithms, the model ensures more reliable and efficient diagnosis.

Key Technologies Used

Technology Purpose
Random Forest Classifies liver disease stages based on multiple decision trees.
K-Nearest Neighbors (KNN) Predicts disease likelihood by comparing patient data with similar cases.
Data Preprocessing Cleans and normalizes input data to improve model performance.
Exploratory Data Analysis (EDA) Identifies patterns and relationships within the dataset.
API Model Enables integration with external healthcare systems.
